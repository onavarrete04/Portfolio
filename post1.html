Tipo de Datos

En los datos pueden haber diferentes tipos de variables que debemos tener en cuenta para aplicar procesos estadísticos, por ejemplo, el índice de masa corporal puede ser un valor al que se le genere un promedio. Sin embargo, hay tipos de datos en los que no sería intuitivo generar este método estadístico.

Pensemos en un tipo de datos que indique la raza, o estado civil, o género y que se codifiquen numéricamente, no podría tener un promedio porque el valor no sería realmente concluyente, entendiendo que para cada raza hay una codificación diferente, para cada género, o para cada estado civil, es decir, si en el conjunto de datos de raza aplicaramos el promedio como método estadístico para toda la información, no sería relevante porque se está desconociendo la codificación y se están tomando más de una variable para el proceso (afro, indígena, mulato, blanco, ot ), sin embargo, para el tipo de datos de raza (categóricos o cualitativos) se pueden aplicar otros métodos de análisis, pero es necesario tener en cuenta estas disertaciones de manera previa para construir de manera adecuada los instrumentos necesarios.


Hay que tener en cuenta que para el dato de la raza, hay algunos métodos de análisis como el tomar el promedio de variable categórica / cualitativa.

Variables cuantitativas

Es una cantidad numérica medible que permite realizar fácilmente operaciones aritméticas, por ejemplo, el índice de masa corporal.

Tipo de variables cuantitativas

Cuantitativas continuas

Son variables que podría tomar cualquier valor dentro de un intervalo de datos y siendo muchos muchos valores posibles, como un gran tabla de índice de masa corporal, o el peso, la altura, el tiempo que toma determinada actividad (correr, saltar, hacer una tarea) que están ligados a procesos continuos.

Cuantitativas Discretas

Es un número finito contable, por ejemplo, número de niños en un hogar, reconociendo que no vamos a tener un número de hijos 2.5, o 2.3, sino siempre son números enteros, también podría ser número de reuniones en un día.

Variables categóricas o cualitativas

Estas variables permiten clasificar individuos o elementos en diferentes grupos, teniendo en cuenta características similares o iguales, por ejemplo el género y nivel de escolaridad.

Tipos de variables categóricas o cualitativas

Ordinal

Tienen un tipo de orden o clasificación asociado, por ejemplo, número de clasificación en la universidad; primer semestre, segundo semestre, octavo semestre, etc. o en los trabajos podría clasificarse según experiencia, juniors o senior.

Nominal

No tienen ninguna clasificación asociada, por ejemplo, raza, género, estado civil, grupo etario que representa mayores de edad y menores de edad.

Casos

1. La edad de una persona es representada como un valor entero normalmente, por lo que podríamos pensar que es un tipo de variable cuantitativa discreta, pero como se establece con un valor relacionado con el tiempo; algunos investigadores les interesaría utilizar este tipo de valor como cuantitativo continuo, entendiendo que en algunos países como EE.UU que para poder conducir se deben tener como mínimo 14 años con 9 meses de edad , por lo cual. para un tipo de investigación que involucre un estudio de edad en el que se identifica desde que momento las personas inician sus clases de conducción, lo más apropiado sería establecer la edad como un valor cuantitativo continuo.


2. En una investigación se desea saber si una persona es mayor de edad o menor de edad, y si bien la edad puede ser representada como un número entero, podemos codificar la información colocando 0 cuando son menores de edad y 1 cuando son mayores de edad, por lo anterior, convertiremos este dato en información categórica o cualitativa, pero se considera variable nominal cualitativa, porque puede codificarse también como A (mayor) y M (menor), teniendo en cuenta que no tienen ninguna clasificación asociada.
